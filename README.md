# Awesome Bonfires

A curated list of projects, tools, and experiments built on [Bonfires AI](https://bonfires.ai) — the knowledge graph platform for agentic workflows.

---

## Core Platform

| Project | Description |
|---------|-------------|
| [TnT (Delve)](https://github.com/NERDDAO/TnT) | Backend API — episodic memory, knowledge graphs, agent stacks, MCP server |
| [bonfires-ai](https://github.com/NERDDAO/bonfires-ai) | Main Bonfires AI service |
| [graphiti](https://github.com/NERDDAO/graphiti) | Real-time knowledge graphs for AI agents (Graphiti fork) |

## Frontends

| Project | Description |
|---------|-------------|

| [bonfires-landing-page](https://github.com/NERDDAO/bonfires-landing-page) | Landing and mint page |
| [synthesis-frontend](https://github.com/NERDDAO/synthesis-frontend) | Synthesis feed — hyperblog aggregator with track filtering |

## CLI & Developer Tools

| Project | Description |
